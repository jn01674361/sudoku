## Build from source

1. git clone ...
2. cd sudoku
3. java -jar build/sudoku.jar -t
4. cd build
5. jar -cvfe sudoku.jar com.sudoku.SudokuGUI *
6. java -jar sudoku.jar -t

## Run prebuilt jar file

1. git clone ...
2. cd sudoku/build
3. java -jar sudoku.jar -t
